## File Ownership

Taken from https://help.ubuntu.com/community/FilePermissions

Make read, write, execute for everyone.
```bash
sudo chmod 777 <filename>
```


## Make executable
```bash
chmod +x <name-of-script>
```


## File Permissions
```bash
chmod u+rwx <filename>
```

- u is for user
- g is for group
- o is for other
- a is for all
